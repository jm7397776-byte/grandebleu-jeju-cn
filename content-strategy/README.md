# 그랑블루 외국인 블로그 — 콘텐츠 다양화 전략

> **목적:** 매일 올라가는 글이 "같은 상품(그랑블루 60분 석양 요트)을 표현만 바꿔 반복"하는 패턴을
> 깨고, **검색 의도(질문)마다 글 1개**가 되도록 만들어 구글이 각 글을 따로 색인하게 한다.

## 왜 지금 색인이 안 되나 (데이터)

중국어 미러 34개 글을 측정한 결과:

- 본문 텍스트는 글마다 **약 90% 다름** (복사 아님 ✅)
- 그러나 핵심 셀링포인트는 **34개 글 전부 동일**: `한국유일`, `쌍동선`, `대포항`, `석양`, `인증`, `Klook` 모두 34/34
- → 글자는 달라도 **전부 같은 검색어("제주 그랑블루 석양 요트")를 노림**
- → 구글은 그중 1~2개만 색인하고 나머지는 "대체 페이지"로 보류 (= 받으신 이메일)

**핵심:** "표현"이 아니라 **"검색 의도"** 를 다르게 해야 한다.

---

## 글 작성 규칙 (생성기/프롬프트에 반영)

각 글은 **하나의 검색 질문**에만 답한다. 한 글에 모든 셀링포인트를 다 넣지 않는다.

1. **H1(제목)** = 사람이 실제로 검색하는 문장. (예: "제주 요트 우천 시 환불되나요?")
2. **도입부 3문장** = 그 질문/상황에만 집중 (다른 글과 겹치지 않게).
3. **본문** = 그 의도에 필요한 정보만 깊게. 공통 사실(가격·정원·안전)은 **필요할 때만, 짧게** 인용.
4. **FAQ 3개** = 그 주제에 특화된 질문 (글마다 달라야 함).
5. **canonical/hreflang** = 자기 자신. 같은 주제 글을 또 만들지 않는다(중복 방지).
6. **내부 링크** = 관련 다른 의도의 글로 2~3개 연결 (같은 주제 글 말고).

## 운영 팁

- **하루 1개**라면 위 60개로 **두 달치**가 확보됨. 클러스터를 번갈아 쓰면 같은 의도가 연달아 나오지 않음.
- 60개를 다 쓰면, 각 글의 **댓글/질문/실제 후기**를 바탕으로 새 의도를 계속 발굴.
- 같은 주제를 또 쓰고 싶으면, 새 글 대신 **기존 글을 업데이트**(날짜 갱신)하는 게 SEO에 더 유리.
- 이 60개는 **언어 공통**이다. 영어/일본어/한국어/중국어 사이트가 각자 자기 언어로 같은 주제를 쓰면, 언어별로 각각 색인된다(서로 중복 아님 — hreflang으로 연결).
